TROUBLESHOOTING:

   Jika masih ada error:

   1. Check PHP Syntax
      php -l bootstrap.php
      php -l functions.php
      php -l admin.php
      php -l login.php
      php -l index.php

   2. Check File Permissions
      • Ensure logs/ directory writable
      • Ensure data/ directory readable
      • Ensure .env file readable

   3. Check Logs
      tail logs/auth.log
      tail logs/audit.log
      tail logs/app.log

   4. Clear Browser Cache
      • Hard refresh (Ctrl+Shift+Delete)
      • Try in incognito mode
      • Clear cookies

   5. Check Apache Error Log
      C:\laragon\logs\apache_error.log
