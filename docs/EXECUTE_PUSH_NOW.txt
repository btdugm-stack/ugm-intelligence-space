### STEP 5: Authentication

When prompted for password:

**If using HTTPS (PAT):**
- GitHub: Create token at https://github.com/settings/tokens
- GitLab: Create token at https://gitlab.com/-/user_settings/personal_access_tokens
- Username: Your GitHub/GitLab username (or email for GitLab)
- Password: Paste the token you created

**If using SSH:**
- Configure SSH key first
- Use SSH URL (starts with git@)

## 🆘 TROUBLESHOOTING

### Error: "Repository not found"
**Solution:**
1. Verify URL is correct (copy again from platform)
2. Check if repository was created
3. Verify internet connection
4. Try again with correct URL

### Error: "Authentication failed"
**Solution:**
1. Use Personal Access Token (not password)
2. GitHub token: https://github.com/settings/tokens
3. GitLab token: https://gitlab.com/-/user_settings/personal_access_tokens
4. Copy token and paste as password

### Error: "fatal: remote origin already exists"
**Solution:**
```bash
git remote remove origin
git remote add origin <YOUR_URL>
git push -u origin main
```

### Error: "Please make sure you have the correct access rights"
**Solution:**
1. Check if repository exists
2. Verify you have access
3. Try using PAT instead of password
4. Check if SSH key is configured
