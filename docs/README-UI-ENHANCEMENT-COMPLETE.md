## 💡 Quick Start

### To Test Locally
1. Open `http://localhost/ugm-intelligence-space-poc/`
2. Test search functionality (hero + catalog)
3. Test all filter combinations
4. Test responsive design (resize browser)
5. Test on mobile device

### To Deploy Changes
1. Pull latest from main branch
2. CSS auto-loads from `assets/style.css`
3. No build process needed
4. Clear browser cache if needed

### To Customize
1. Edit CSS variables in `:root` for colors
2. Edit font sizes in `h1-h6` for typography
3. Modify grid columns in media queries for layout
4. Update footer links in `index.php`
