## 🔐 Authentication

### GitHub Authentication
- **HTTPS:** Use GitHub Personal Access Token
  ```
  1. Go to https://github.com/settings/tokens
  2. Click "Generate new token"
  3. Select "repo" scope
  4. Copy token
  5. Use token as password when prompted
  ```
- **SSH:** Use SSH key (more secure for frequent pushes)

### GitLab Authentication
- **HTTPS:** Use GitLab Personal Access Token
  ```
  1. Go to https://gitlab.com/-/user_settings/personal_access_tokens
  2. Create new token with "api" scope
  3. Copy token
  4. Use token as password when prompted
  ```

## 🆘 Troubleshooting

### Issue: "Repository not found"
```
❌ Solution:
1. Check if repository URL is correct
2. Verify you created the repo on GitHub/GitLab
3. Check if you have internet connection
4. Try using PAT (Personal Access Token) instead of password
```

### Issue: "fatal: remote origin already exists"
```
❌ Solution:
git remote remove origin
git remote add origin YOUR_URL_HERE
git push -u origin main
```

### Issue: "Authentication failed"
```
❌ Solution:
1. GitHub: Use Personal Access Token (not password)
2. GitLab: Use Personal Access Token
3. Or: Configure SSH keys for authentication
```

### Issue: "Permission denied"
```
❌ Solution:
1. Verify you have push access to repository
2. Try adding SSH key to your GitHub/GitLab account
3. Check if repository is set to Private (you may not have access)
```
