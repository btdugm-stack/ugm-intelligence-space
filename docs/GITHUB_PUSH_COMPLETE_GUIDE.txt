## 📋 COMPLETE GITHUB PUSH GUIDE

### STEP 1: Create GitHub Repository (5 minutes)

**1.1 Go to GitHub**
```
URL: https://github.com/new
```

**1.2 Fill in Repository Form**
```
Repository name:  ugm-intelligence-space
Description:      Dashboard Gateway & Decision Intelligence Portal
Visibility:       Public (recommended) or Private
```

**1.3 Click "Create repository"**

**1.4 You will see a page with instructions**
Look for the section: "...or push an existing repository from the command line"

**1.5 Copy the HTTPS URL**
```
It will look like:
https://github.com/YOUR_USERNAME/ugm-intelligence-space.git

Save this URL! You'll need it.
```

═══════════════════════════════════════════════════════════════════════════════

### STEP 2: Get Personal Access Token (3 minutes)

**2.1 Go to GitHub Settings**
```
URL: https://github.com/settings/tokens
```

**2.2 Click "Generate new token"**

**2.3 Select Scope**
```
Check: ☑ repo (Full control of private repositories)
```

**2.4 Click "Generate token"**

**2.5 IMPORTANT: Copy the token!**
```
The token looks like:
ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxX

You won't be able to see it again!
Save it somewhere safe.
```

═══════════════════════════════════════════════════════════════════════════════

### STEP 3: Execute Push Commands

**3.1 Open Command Prompt**

Windows 10/11:
```
Press: Win + X
Choose: Windows PowerShell or Command Prompt
```

Or:
```
Press: Win + R
Type: cmd
Press: Enter
```

**3.2 Navigate to Project Directory**

```bash
cd c:\laragon\www\ugm-intelligence-space-poc
```

Then press Enter.

**3.3 Add Remote Origin**

Replace YOUR_USERNAME with your actual GitHub username:

```bash
git remote add origin https://github.com/YOUR_USERNAME/ugm-intelligence-space.git
```

Example:
```bash
git remote add origin https://github.com/john-doe/ugm-intelligence-space.git
```

Then press Enter.

**3.4 Set Main Branch**

```bash
git branch -M main
```

Then press Enter.

**3.5 Push to GitHub**

```bash
git push -u origin main
```

Then press Enter.

═══════════════════════════════════════════════════════════════════════════════

### STEP 4: Authentication with Personal Access Token

**When Terminal Asks for Password:**

```
Username for 'https://github.com': YOUR_USERNAME
Password for 'https://github.com/YOUR_USERNAME': [PASTE TOKEN HERE]
```

**What to Do:**

1. Type your GitHub username and press Enter
2. When asked for password:
   - Right-click terminal
   - Choose "Paste"
   - Paste the Personal Access Token
   - Press Enter

**Important:** The password won't show as you type. This is normal!

## 🆘 TROUBLESHOOTING

### Error: "fatal: remote origin already exists"

**Cause:** Origin already added

**Solution:**
```bash
git remote remove origin
git remote add origin https://github.com/YOUR_USERNAME/ugm-intelligence-space.git
git push -u origin main
```

───────────────────────────────────────────────────────────────────────────────

### Error: "Authentication failed for 'https://github.com/...'"

**Cause:** Wrong token or username

**Solution:**
1. Generate new token: https://github.com/settings/tokens
2. Try pushing again
3. Use token (not password) when prompted

───────────────────────────────────────────────────────────────────────────────

### Error: "Repository not found"

**Cause:** Repository doesn't exist or URL is wrong

**Solution:**
1. Verify URL is correct (copy from GitHub again)
2. Check if repository was actually created
3. Try again with correct URL

───────────────────────────────────────────────────────────────────────────────

### Error: "Please make sure you have the correct access rights"

**Cause:** Permission issue or SSH key problem

**Solution:**
1. Use HTTPS instead of SSH
2. Make sure token has "repo" scope
3. Try with new token
