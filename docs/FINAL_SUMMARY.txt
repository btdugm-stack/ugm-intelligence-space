🔐 AUTHENTICATION SETUP

GitHub (Recommended):
  1. Go to https://github.com/settings/tokens
  2. Generate new Personal Access Token
  3. Select "repo" scope
  4. Copy token
  5. Use as password when git asks

GitLab:
  1. Go to https://gitlab.com/-/user_settings/personal_access_tokens
  2. Create new token with "api" scope
  3. Copy token
  4. Use as password when git asks

SSH (More Secure):
  • Configure SSH keys on your platform
  • Use SSH URLs instead of HTTPS
  • No password required for future pushes
