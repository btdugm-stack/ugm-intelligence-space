## 📋 FOLLOW THESE STEPS EXACTLY IN ORDER:

### STEP 1️⃣  CREATE GITHUB REPOSITORY (5 minutes)

1. Open browser
2. Go to: https://github.com/new
3. Fill form:
   - Repository name: ugm-intelligence-space
   - Description: Dashboard Gateway & Decision Intelligence Portal
   - Public (recommended)
4. Click "Create repository"
5. Copy the HTTPS URL (looks like):
   https://github.com/YOUR_USERNAME/ugm-intelligence-space.git

📌 SAVE THIS URL!

═══════════════════════════════════════════════════════════════════════════════

### STEP 2️⃣  GET PERSONAL ACCESS TOKEN (3 minutes)

1. In new browser tab, go to: https://github.com/settings/tokens
2. Click "Generate new token"
3. Check ☑ repo (Full control of private repositories)
4. Click "Generate token"
5. Copy the token (starts with "ghp_")

⚠️  IMPORTANT: Copy it NOW! You won't see it again!

═══════════════════════════════════════════════════════════════════════════════

### STEP 3️⃣  OPEN TERMINAL

1. Press: Win + X
2. Choose: Windows PowerShell or Command Prompt

(Or: Press Win + R, type "cmd", press Enter)

═══════════════════════════════════════════════════════════════════════════════

### STEP 4️⃣  EXECUTE THESE COMMANDS ONE BY ONE

⚠️  REPLACE YOUR_USERNAME with your GitHub username!
⚠️  Replace YOUR_REPO_URL with the URL you copied in Step 1!

─────────────────────────────────────────────────────────────────────────────

COMMAND 1 - Navigate to project:
─────────────────────────────────

cd c:\laragon\www\ugm-intelligence-space-poc

Press Enter

─────────────────────────────────────────────────────────────────────────────

COMMAND 2 - Add remote origin:
──────────────────────────────

git remote add origin https://github.com/YOUR_USERNAME/ugm-intelligence-space.git

Replace YOUR_USERNAME with your GitHub username!

Example:
git remote add origin https://github.com/john-doe/ugm-intelligence-space.git

Press Enter

─────────────────────────────────────────────────────────────────────────────

COMMAND 3 - Set branch to main:
───────────────────────────────

git branch -M main

Press Enter

─────────────────────────────────────────────────────────────────────────────

COMMAND 4 - PUSH TO GITHUB:
──────────────────────────

git push -u origin main

Press Enter

⏳ Wait for it to complete...

─────────────────────────────────────────────────────────────────────────────

### STEP 5️⃣  AUTHENTICATION

When terminal asks:
```
Username for 'https://github.com': 
```

1. Type your GitHub username
2. Press Enter

When it asks:
```
Password for 'https://github.com/YOUR_USERNAME':
```

1. Right-click terminal
2. Choose "Paste"
3. Paste the Personal Access Token (from Step 2)
4. Press Enter

⚠️  Password won't show as you type - this is normal!

## ❌ IF SOMETHING GOES WRONG

Error: "fatal: remote origin already exists"
→ Run: git remote remove origin
→ Try again

Error: "Authentication failed"
→ Use Personal Access Token (not password)
→ Get new token from https://github.com/settings/tokens

Error: "Repository not found"
→ Check URL is correct
→ Verify repository was created on GitHub
