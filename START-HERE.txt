❓ COMMON QUESTIONS
════════════════════════════════════════════════════════════════

Q: Will my current project be affected?
A: No! Deployment only adds Git tracking. Your project works.

Q: What if something goes wrong?
A: Remove .git folder and retry. Your files are safe.

Q: How do I undo the deployment?
A: Keep working locally. Delete .git if needed. Always reversible.

Q: Can I use HTTPS instead of SSH?
A: Yes! Use Personal Access Token (PAT) when prompted.

Q: How do I update after deploying?
A: Use standard Git commands:
   git add .
   git commit -m "message"
   git push origin main

Q: What if I made a mistake?
A: See "Troubleshooting" in DEPLOYMENT-GUIDE.md
